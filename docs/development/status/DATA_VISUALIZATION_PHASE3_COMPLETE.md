## 📈 数据可视化功能详情

### 支持的图表类型
1. **线图 (LineChart)**: 时间序列趋势分析
2. **柱状图 (BarChart)**: 分类数据对比
3. **饼图 (PieChart)**: 比例分布展示
4. **面积图 (AreaChart)**: 累计趋势分析
5. **雷达图 (RadarChart)**: 多维评估分析
6. **数据表格 (DataTable)**: 结构化数据展示

### 数据维度
1. **时间维度**: 日度、周度、月度数据
2. **分类维度**: 项目类型、成本构成、风险类型
3. **数值维度**: 成本、进度、风险、效率
4. **关系维度**: 预算vs实际、估算vs实际

### 交互功能
1. **时间筛选**: 切换日/周/月视图
2. **数据刷新**: 手动刷新数据
3. **图表缩放**: 响应式缩放
4. **工具提示**: 详细数据提示
5. **图例控制**: 显示/隐藏数据系列
