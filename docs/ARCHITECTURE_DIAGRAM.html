Enterprise RAG System - Six-Layer Layered Architecture

L1 User Interface Layer (UI Layer)
▼
Provides user interaction interfaces including Web UI, management interface, and external API clients
📊 Streamlit Web UI
Main user interface providing query, visualization, and management features
Streamlit JavaScript
🔧 Prompt Admin UI
Prompt management interface with version control and A/B testing
React TypeScript
🔌 External API Client
RESTful API supporting third-party system integration
REST OpenAPI
L2 API Gateway Layer
▼
Unified API entry point responsible for route distribution, authentication, request validation, and caching strategy
🚦 FastAPI Routers
Unified API route management supporting workflow queries, cost estimation, and prompt management
FastAPI Pydantic
🔐 Authentication & Authorization
JWT token validation, tenant-level data isolation, and RBAC permission control
JWT OAuth2
✅ Input Validation
Pydantic model validation for request parameters ensuring data integrity and security
Pydantic Validator
⚡ Query Cache
Redis cache layer to reduce duplicate query costs and improve response speed
Redis Cache
L3 Business Services Layer
▼
Core business logic layer responsible for workflow orchestration, intent recognition, intelligent routing, and budget control
🔄 Workflow Orchestrator
Manages complex query execution flows using LangGraph with state machine and conditional routing
LangGraph State Machine
🎯 Intent Classifier
Identifies user query types (RAG/Code Analysis/Hybrid LLM) to determine execution path
NLP Classification
💬 Prompt Selector
Dynamically selects optimal prompt templates with version control and A/B testing support
Template Engine A/B Testing
🧭 Routing & Budget Policy
Intelligent routing decisions based on cost, performance, and budget with real-time monitoring
Policy Engine Budget Control
L4 AI Runtime Layer
▼
AI capability core layer providing RAG, LLM dispatch, code execution, and cost estimation capabilities
🔍 RAG Engine
Hybrid retrieval system supporting semantic retrieval, keyword retrieval, and intelligent reranking
LangChain pgvector BM25
🤖 LLM Dispatch Service
Multi-provider LLM dispatch supporting OpenAI, Anthropic, Cohere, with intelligent failover
OpenAI Anthropic Cohere
⚙️ Code Execution Manager
Docker container isolated code execution environment supporting Python, R, and other languages
Docker Sandbox Timeout Control
💰 Cost Estimation Service
Machine learning based cost prediction model providing accurate cost estimation and confidence intervals
scikit-learn Linear Regression
L5 Data Storage Layer
▼
Data persistence layer supporting relational data, vector data, model artifacts, and file storage
🗄️ PostgreSQL + pgvector
Primary database with vector retrieval support containing documents, prompts, usage, budgets, and other tables
PostgreSQL 15+ pgvector HNSW Index
📦 Model Artifacts
Versioned storage for cost estimation models, embedding models, and other model artifacts
File System Version Control
📄 Document Store
Storage for document chunks, embedding vectors, and metadata with efficient retrieval
Vector Store Embeddings
L6 Security & Platform Layer
▼
Cross-layer security protection, observability, audit logging, and operational support
🛡️ Data Redaction
Automatic PII identification and redaction to protect sensitive data
Regex NER
🚪 Egress Guard
External API call monitoring and filtering to prevent data leakage
Firewall Rate Limiting
📊 Observability
Metrics monitoring, log aggregation, and distributed tracing
Prometheus Grafana ELK
📝 Audit Logging
Complete operational audit trail supporting compliance requirements
Audit Log Compliance
