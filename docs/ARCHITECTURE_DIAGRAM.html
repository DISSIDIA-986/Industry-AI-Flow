Industry AI Flow

企业级RAG系统 - 六层分层架构

L1 用户界面层 (UI Layer)
▼
提供用户交互界面，包括Web UI、管理界面和外部API客户端
📊 Streamlit Web UI
主用户界面，提供查询、可视化、管理功能
Streamlit JavaScript
🔧 Prompt Admin UI
提示词管理界面，支持版本控制和A/B测试
React TypeScript
🔌 External API Client
RESTful API，支持第三方系统集成
REST OpenAPI
L2 API网关层 (API Gateway Layer)
▼
统一API入口，负责路由分发、身份认证、请求验证和缓存策略
🚦 FastAPI Routers
统一的API路由管理，支持工作流查询、成本估算、提示词管理等
FastAPI Pydantic
🔐 Authentication & Authorization
JWT token验证，租户级别数据隔离和RBAC权限控制
JWT OAuth2
✅ Input Validation
Pydantic模型验证请求参数，确保数据完整性和安全性
Pydantic Validator
⚡ Query Cache
Redis缓存层，减少重复查询成本，提升响应速度
Redis Cache
L3 业务服务层 (Business Services Layer)
▼
核心业务逻辑层，负责工作流编排、意图识别、智能路由和预算控制
🔄 Workflow Orchestrator
使用LangGraph管理复杂的查询执行流程，支持状态机和条件路由
LangGraph State Machine
🎯 Intent Classifier
识别用户查询类型（RAG/代码分析/混合LLM），决定执行路径
NLP Classification
💬 Prompt Selector
动态选择最优提示词模板，支持版本控制和A/B测试
Template Engine A/B Testing
🧭 Routing & Budget Policy
基于成本、性能和预算的智能路由决策，实时预算监控和控制
Policy Engine Budget Control
L4 AI运行时层 (AI Runtime Layer)
▼
AI能力核心层，提供RAG、LLM调度、代码执行和成本估算能力
🔍 RAG Engine
混合检索系统，支持语义检索、关键词检索和智能重排序
LangChain pgvector BM25
🤖 LLM Dispatch Service
多提供商LLM调度，支持OpenAI、Anthropic、Cohere等，智能故障转移
OpenAI Anthropic Cohere
⚙️ Code Execution Manager
Docker容器隔离的代码执行环境，支持Python、R等多种语言
Docker Sandbox Timeout Control
💰 Cost Estimation Service
基于机器学习的成本预测模型，提供准确的成本估算和置信区间
scikit-learn Linear Regression
L5 数据存储层 (Data Storage Layer)
▼
数据持久化层，支持关系数据、向量数据、模型工件和文件存储
🗄️ PostgreSQL + pgvector
主数据库，支持向量检索和复杂查询，包含文档、提示词、用量、预算等表
PostgreSQL 15+ pgvector HNSW Index
📦 Model Artifacts
成本估算模型、嵌入模型等模型工件的版本化存储
File System Version Control
📄 Document Store
文档块、嵌入向量和元数据的存储，支持高效检索
Vector Store Embeddings
L6 安全与平台层 (Security & Platform Layer)
▼
跨层安全防护、可观测性、审计日志和运维支持
🛡️ Data Redaction
PII信息自动识别和脱敏，保护敏感数据
Regex NER
🚪 Egress Guard
外部API调用监控和过滤，防止数据泄露
Firewall Rate Limiting
📊 Observability
指标监控、日志聚合、分布式追踪
Prometheus Grafana ELK
📝 Audit Logging
完整的操作审计追踪，支持合规要求
Audit Log Compliance

📊 请求处理流程

用户请求 (L1 - UI Layer)
API网关处理 (L2 - 认证、验证、路由)
业务服务处理 (L3 - 意图识别、工作流编排)
AI执行 (L4 - RAG/LLM/代码执行)
数据访问 (L5 - 数据库查询、模型推理)
安全处理 (L6 - 脱敏、审计、监控)
返回结果 (L1 - 展示给用户)
