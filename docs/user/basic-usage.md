# 基础使用指南

欢迎使用 Industry AI Flow！本指南将帮助您快速上手系统的基本功能。

## 🚀 快速开始

### 1. 启动应用
```bash
# 确保在项目根目录
cd Industry-AI-Flow

# 激活虚拟环境
source backend/venv_llamacpp/bin/activate  # Linux/Mac
# 或 backend\venv_llamacpp\Scripts\activate  # Windows

# 启动 Streamlit 应用
streamlit run streamlit_app.py
```

### 2. 访问界面
- 打开浏览器访问 `http://localhost:8501`
- 您将看到 Industry AI Flow 的主界面

## 📱 界面介绍

### 侧边栏配置
1. **模型设置**
   - 选择模型类型（llama.cpp、OpenAI、Claude等）
   - 调整 Temperature（创造性参数）
   - 设置 Max Tokens（回答长度限制）

2. **文档上传**
   - 支持格式：PDF、DOCX、TXT、JPG、PNG
   - 可同时上传多个文档
   - 系统会自动解析和索引

3. **系统状态**
   - 显示当前模型状态
   - 系统运行指标
   - 错误提示信息

### 主聊天界面
1. **对话区域**
   - 显示用户问题和AI回答
   - 支持markdown格式显示
   - 消息时间戳

2. **输入框**
   - 底部文本输入框
   - 支持多行输入
   - Enter 键发送消息

## 📄 文档处理

### 支持的文档格式
- **文本文件**：TXT, MD
- **办公文档**：PDF, DOCX, DOC
- **图像文件**：JPG, PNG, JPEG, BMP
- **其他格式**：HTML, XML

### 文档上传步骤
1. 在侧边栏找到"文档上传"区域
2. 点击"浏览文件"或拖拽文件到上传区域
3. 选择要上传的文件（支持多选）
4. 等待上传和处理完成
5. 系统会显示处理结果

### OCR 文字识别
- **图像文档**：系统自动使用 PaddleOCR 进行文字识别
- **识别精度**：支持中英文混合识别
- **处理时间**：根据文档大小，通常需要几秒到几十秒

## ⚙️ 个性化设置

### 调整回答风格
通过修改 Temperature 参数：
- **低值（0.1-0.3）**：更保守、准确的回答
- **中值（0.7）**：平衡创造性和准确性（推荐）
- **高值（0.8-1.0）**：更有创造性但可能不够准确

### 控制回答长度
通过 Max Tokens 参数：
- **短回答**：500-1000 tokens
- **中等回答**：1500-2000 tokens（推荐）
- **长回答**：3000+ tokens

## 🛠️ 故障排除

### 常见问题

**Q: 文档上传失败**
A: 检查文件格式是否支持，文件大小是否超限（建议<50MB）

**Q: 回答生成很慢**
A: 尝试调整模型参数，或检查网络连接

**Q: OCR识别不准确**
A: 确保图像清晰，文字大小适中，避免复杂背景

**Q: 问答不相关**
A: 尝试重新上传文档，或调整问题的表述方式

### 错误处理
- 系统会显示具体的错误信息
- 查看侧边栏的系统状态
- 如问题持续，请查看故障排除指南
