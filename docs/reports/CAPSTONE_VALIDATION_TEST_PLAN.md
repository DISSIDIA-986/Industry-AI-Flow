### 2.3 数据流验证

```

特征工程（提取数值特征、编码类别特征）→

```

### 2.4 边界条件测试

- [ ] **缺失特征**: 部分特征值为空的预测
- [ ] **异常值**: 极端面积、极高成本
- [ ] **新项目类型**: 训练集中未见过的项目类型
- [ ] **小数据集**: 训练集少于50条记录
- [ ] **数据不平衡**: 某类项目样本过少
