**Housing 数据集概况**:

- **特征数量**: 13 个特征

- **缺失值**: 0个 (数据质量良好)

**关键统计指标**:
- **价格范围**: 1,750,000 - 13,300,000
- **平均价格**: 4,766,729
- **平均面积**: 5,151 sq.ft
- **面积-价格相关性**: 0.536 (中等正相关)

- **R² 分数**: 0.612 (模型解释了61.2%的价格变异)
- **平均绝对误差 (MAE)**: 1,025,290
- **均方根误差 (RMSE)**: 1,401,263
- **训练集大小**: 436条记录

**特征重要性排名**:

4. **parking**: 0.057 (停车位)
5. **stories**: 0.055 (楼层数)

5. **特征重要性图** - Top 10特征的条形图可视化

### 智能洞察生成

系统成功生成了 **5个关键数据洞察**:

1. **价格分层分析**: 经济型房源平均价格 2,853,318，豪华型房源平均价格 7,309,510

3. **地段价值**: 主路房源相比非主路房源有 146.9% 的溢价
4. **装修价值**: 精装修房相比毛坯房有 136.9% 的溢价
