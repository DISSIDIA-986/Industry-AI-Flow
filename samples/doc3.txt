Ollama本地大语言模型

Ollama是一个用于运行本地大语言模型的工具。

支持的模型：
- Llama 2：Meta开源模型
- Mistral：高性能开源模型
- Qwen：阿里巴巴千问系列
- Code Llama：代码生成专用
- Vicuna：基于Llama微调

Qwen2.5特点：
- 多语言支持，中文表现优秀
- 7B参数量，适合本地运行
- 推理速度快
- 上下文长度可达32K

安装使用：
1. 下载Ollama客户端
2. 运行: ollama pull qwen2.5:7b
3. 启动服务: ollama serve
4. API调用: http://localhost:11434

内存需求：
- 7B模型约需8GB RAM
- 建议16GB以上内存
- 支持GPU加速
